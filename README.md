# 热榜助手

## 应用简介
热榜助手是一款聚合类 Android 应用，旨在为用户提供各大主流平台（如微博、知乎、百度、抖音等）的实时热搜榜单。用户可以方便地查看不同平台的热点话题，支持收藏、渠道切换及跳转至原平台查看。

## 主要功能

### 实时热搜聚合
- 支持多个主流平台的热搜榜单
- 实时更新热点话题
- 热度值展示

### 收藏管理
- 本地持久化收藏列表
- 按收藏时间倒序展示
- 支持按来源平台过滤
- 支持按时间范围筛选

### 渠道切换
- 横向滑动的平台切换栏
- 仅展示用户在设置中开启的平台
- 点击即时刷新当前列表数据

### 浏览器跳转
- 尝试唤起对应App打开链接
- 应用内展示(Chrome Custom Tabs)
- 系统浏览器兜底

### 主题切换
- 支持浅色/深色模式
- 手动切换主题
- 主题偏好持久化

## 技术特点

### 架构模式
- MVVM架构
- 模块化设计
- 数据仓库模式

### 技术栈
- 开发语言: Java
- UI布局: XML (View-based)
- 网络请求: Retrofit 2 + OkHttp 3 + Gson
- 数据持久化: Room Database + SharedPreferences
- 刷新组件: SmartRefreshLayout
- 列表展示: RecyclerView
- 浏览器跳转: Android Intent / Chrome Custom Tabs
