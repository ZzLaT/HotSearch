# Android 面试高频问题（按重要性从高到低）

## 1. RecyclerView（列表容器）— 20 题
1. RecyclerView 的复用机制是什么？和 ListView 的复用机制核心差异在哪里？
2. ViewHolder 为什么要做成静态内部类？不做会有什么风险？
3. RecyclerView 的预取（prefetch）/ 预布局（pre-layout）在什么场景会触发？对性能有什么影响？
4. notifyDataSetChanged 与 notifyItemInserted/Removed/Changed 的差别是什么？各自的性能与动画表现如何？
5. DiffUtil 的工作原理是什么？如何避免计算 Diff 时卡顿（例如大列表）？
6. ListAdapter/AsyncListDiffer 与手写 DiffUtil 的差异与适用场景是什么？
7. setHasStableIds(true) 与 getItemId 的作用是什么？会对动画、复用、闪烁产生什么影响？
8. RecyclerView 的 ItemAnimator 为什么容易导致闪烁/错位？如何关闭或定制来避免问题？
9. RecyclerView 的嵌套滑动（NestedScrolling）机制是什么？嵌套 RecyclerView/CoordinatorLayout 时常见冲突怎么解决？
10. LinearLayoutManager/GridLayoutManager/StaggeredGridLayoutManager 的差异与典型坑有哪些？
11. StaggeredGridLayoutManager 为什么会“跳动/错位/空洞”？如何配置 gap strategy 或改用其他方案？
12. 如何在 RecyclerView 中实现多类型（多 ViewType）？如何避免 ViewType 滥用导致的复用失效？
13. 如何处理图片加载（Glide/Coil）在快速滑动时的错位与重复请求？如何做请求取消/占位与回收？
14. RecyclerView 的缓存层级（Recycler、Cache、RecycledViewPool）分别是什么？如何共享 RecycledViewPool 优化多列表页面？
15. 如何实现分页加载（上拉加载更多）并避免重复触发与“加载中”状态错乱？
16. 如何实现局部刷新（payload）？payload 的最佳实践与注意点是什么？
17. 如何正确处理点击事件（防抖、位置变化、adapterPosition/bindingAdapterPosition）避免崩溃与错点？
18. 如何定位 RecyclerView 的卡顿（Choreographer、Systrace/Perfetto、Layout Inspector）？通常瓶颈在什么地方？
19. RecyclerView item 布局层级过深导致性能差，如何优化（ConstraintLayout、merge、减少 overdraw）？
20. RecyclerView 中如何避免内存泄漏（Adapter 持有 Context/Listener、匿名内部类、Handler）？

## 2. Retrofit（网络请求）— 20 题
1. Retrofit 的整体架构是什么？它与 OkHttp 的关系与分工是什么？
2. @GET/@POST/@Body/@Query/@Path/@Field/@Part 等注解分别如何映射到 HTTP 请求？常见用错点是什么？
3. Converter（Gson/Moshi/Proto）与 CallAdapter（RxJava/Coroutine）分别解决什么问题？
4. Retrofit + OkHttp 的拦截器链（Interceptor）工作机制是什么？应用拦截器与网络拦截器区别？
5. 如何做统一的鉴权（Token 注入）与 Token 过期刷新？如何避免并发刷新导致“惊群”？
6. 如何实现请求重试/退避（Exponential Backoff）？哪些错误应该重试，哪些不应该？
7. 如何设置超时（connect/read/write/callTimeout）？为什么只配 readTimeout 可能不够？
8. 如何取消请求？Call.cancel() 在 OkHttp/Retrofit 中的传播链路是什么？
9. 如何处理请求生命周期与页面销毁（Activity/Fragment）绑定，避免回调打到已销毁页面？
10. Retrofit 的同步请求与异步请求分别怎么做？它们分别在哪个线程执行？
11. 如何正确处理 HTTP 错误与业务错误（2xx 但业务 code 非 0）？推荐的错误封装方式是什么？
12. 如何处理 JSON 字段变更/缺失/类型不匹配导致的解析崩溃？有哪些兼容策略？
13. 如何上传文件（Multipart）与下载大文件（Streaming）？如何避免 OOM？
14. 如何打印与脱敏日志（HttpLoggingInterceptor、自定义拦截器）同时避免泄露敏感信息？
15. 如何做缓存（Cache-Control、OkHttp Cache、自定义缓存）？缓存的命中条件与陷阱有哪些？
16. 如何处理 HTTPS（证书校验、证书锁定 pinning）？遇到抓包/中间人攻击如何防护？
17. 如何在 Retrofit 中实现多 BaseUrl 或动态切换域名？有哪些设计方式与风险？
18. 如何做网络层性能优化（连接复用、HTTP/2、DNS、压缩、减少重定向）？
19. 如何做并发请求编排（并行/串行/依赖）并保证可读性与可测试性？
20. 如何做网络层测试（MockWebServer、接口回放、契约测试）与稳定性建设？

## 3. 同步 / 异步 / 并发 — 20 题

6. ThreadLocal 的原理与使用场景是什么？有哪些内存泄漏风险（尤其在 Android）？
7. Java 线程池（Executors/ThreadPoolExecutor）的核心参数如何配置？如何避免 OOM/任务堆积？
8. Handler/Looper/MessageQueue 的工作机制是什么？消息是如何入队与取出的？
9. 主线程为什么不能做耗时操作？严格来说“耗时”是怎么定义/怎么度量的？
10. Handler 引起内存泄漏的根因是什么？如何用静态内部类 + 弱引用或移除消息避免？
11. Future/Callable/CompletableFuture 的使用场景是什么？如何处理取消与超时？
12. Kotlin 协程的结构化并发是什么？为什么推荐用 structured concurrency？
13. Dispatchers.Main/IO/Default 的区别？为什么“IO 适合网络/磁盘”但不是无限线程？
14. suspend 与非 suspend 的边界是什么？挂起函数到底“挂起了什么”？
15. Job/SupervisorJob 的差别？异常传播与取消传播规则是什么？
16. coroutineScope vs supervisorScope 的差别？并发子任务失败时如何做到“部分失败可继续”？
17. Flow 的冷流/热流区别是什么？StateFlow/SharedFlow 的使用场景？
18. backpressure（背压）是什么？在 RxJava/Flow 中如何处理高频事件（例如搜索联想）？
19. 如何避免并发下的重复请求/重复点击（去重、互斥、单飞 singleflight、debounce/throttle）？
20. 如何做并发调试与问题定位（线程 dump、StrictMode、日志关联 traceId）？
